Ryoku's desktop is Hyprland + the Ryoku shell. Hyprland handles compositing,
input, and window layout; the Ryoku shell paints the taskbar, popouts,
dashboard, launcher, notifications, lock-adjacent surfaces, and settings
surfaces.

## Hyprland workspaces

Ryoku ships Hyprland defaults for workspace navigation, window actions, and shell shortcuts.

- `Mod+1` through `Mod+0` jump to workspaces 1-10.
- `Mod+Shift+1` through `Mod+Shift+0` move the focused window to that workspace.
- `Mod+H/J/K/L` or arrow keys move focus.
- `Mod+Shift+H/J/K/L` or shifted arrow keys move the focused window.
- `Mod+F` fullscreens; `Mod+A` toggles floating.

Use [Keybindings](/docs/keybindings) for the full shipped list.

## Taskbar

The taskbar is the persistent desktop frame. In the current shell it owns:

- **OS entry point:** the Ryoku mark and session entry points.
- **Active window:** title and app icon of the focused window.
- **Workspaces:** workspace buttons for the current monitor, including special workspace indicators.
- **Tray:** third-party tray items from apps such as chat, sync, and game clients.
- **Status icons:** audio, microphone, keyboard layout, network, bluetooth, battery, and lock-state indicators when enabled.
- **Clock and power:** time/date, session, and power entry points.

Taskbar behavior is configured from **Settings → Taskbar**.

## Island and Dashboard

The island is the compact media/attention surface near the taskbar. It pairs
with the dashboard for richer status:

- **Media:** active player controls, artwork, and visualizer surfaces.
- **Performance:** CPU, GPU, memory, storage, network, and battery cards when enabled.
- **Weather:** current weather and forecast views when a location is configured.
- **Calendar / date / user cards:** quick context without opening a full app.

## Launcher

Open the launcher with `Mod+Space`. It handles:

- app search and launch;
- calculator input through the launcher command prefix;
- wallpaper selection;
- color scheme and Material variant actions.

## Drawers and Utilities

- **Notifications:** the sidebar stores grouped notification history and actions.
- **Utilities:** quick toggles, idle inhibit, screen recording controls, and a recordings list.
- **OSD:** volume, brightness, media, and other transient feedback.

## Lock screen and login

- `Mod+Alt+L` locks the session. The lock screen shows the time, the active media, and a password field. Unlock with your account password.
- SDDM (the display manager) shipping with Ryoku has switchable themes; pick one from **Settings → Login screen**.

## Where things live

- Hyprland config: `~/.config/hypr/hyprland.conf`.
- Shell config: `~/.config/ryoku/shell.json` (the Ryoku shell reads this; Settings writes to it).
- Wallpapers: `~/Pictures/Wallpapers`.
- Shell token overrides: `~/.config/ryoku/shell-tokens.json`.
- Themes / palettes: generated under `~/.local/state/quickshell/user/generated/`.
- Logs: `journalctl --user -u ryoku-shell` for the shell; use `hyprctl monitors` and the Hyprland user-session logs for compositor state.
