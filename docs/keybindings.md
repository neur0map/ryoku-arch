# Ryoku Keybindings

Source of truth: Hyprland loads shipped binds from `config/hypr/hyprland.conf`.

`Mod` means `Super` on a normal Ryoku install.

## Session And Compositor

| Binding | Action |
| --- | --- |
| `Mod+Shift+R` | Reload Hyprland. |
| `Mod+Shift+E` | Exit Hyprland. |
| `Mod+Alt+L` | Lock session. |
