**力と美のために** — *for the sake of power and beauty.*

Ryoku is a premium Arch workstation for capable desktops and laptops. It is pre-riced, plugin-minded, and tuned for people who want a fast, polished daily machine from first boot.

The desktop is [Hyprland](https://hypr.land/), with a [Quickshell](https://quickshell.outfoxxed.me)-based topbar, sidebars, lock screen, launcher, and control center. Workflow modules (VPN, capture, updates, hardware, media, and developer tools) are integrated as shell-facing lanes instead of random tray-only scripts.

## What ships

- **Desktop**: Hyprland session, Quickshell topbar / sidebars / lock / launcher / control center, switchable themes, shell-first workflows.
- **Plugin lanes**: OpenVPN, Tailscale, capture workflows, hardware controls, and developer integrations.
- **Daily-driver fit**: Development, content work, gaming, media, notes, terminals, and container workflows on strong hardware.
- **Brand**: cohesive Greek-Noir palette, switchable Material 3 skins (`material`, `aurora`, `angel`, `ryoku-shell`, `cards`).

## Scope

Ryoku is built for people who want an expressive Arch workstation instead of spending a day re-wiring everything by hand.

Ryoku is not trying to be a lightweight starter kit. It is intentionally tuned for powerful machines where the visual layer and workflow surface are part of the product.

## Status

Public preview. Signed alpha ISOs are downloadable at [`iso.ryoku.dev`](https://iso.ryoku.dev/stable/); the first tagged stable release is the next milestone.

The preview track is `unstable-dev`; stable releases are prepared on `main`. The shell is in a transition from Omarchy inheritance and Celestia-inspired structure toward a Ryoku-owned plugin shell layer, while keeping installer, command, and runtime behavior coherent.
