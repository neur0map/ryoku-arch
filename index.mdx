---
title: Ryoku
description: An Arch Linux security workstation fork of Omarchy with a Niri-based desktop, Quickshell UI, and a signed offline ISO.
---

Ryoku is an opinionated Arch Linux desktop targeted at security work: a Niri tiling compositor, a custom Quickshell UI, hardened defaults, and a signed offline ISO that boots and installs without network access. It started as a fork of [Omarchy](omarchy-heritage) and has diverged on the desktop, theming, and release pipeline.
