# Security Policy

Ryoku is currently in public preview. Please report security-sensitive issues privately instead of opening a public issue.

## Reporting

Email the maintainer listed on the GitHub profile, or use a private GitHub security advisory if available for this repository. Include:

- affected commit or ISO date
- hardware and install mode, if relevant
- exact reproduction steps
- impact and any suggested mitigation

## Scope

Security reports are welcome for installer behavior, package provenance, privilege boundaries, update flow, bundled scripts, desktop integrations, and default network exposure.
